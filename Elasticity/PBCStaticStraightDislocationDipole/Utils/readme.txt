------------------------------------------------------
This directory contains small utilities (Linux binary)
------------------------------------------------------

T2AbsoluteToObserver:  rank-2 tensor transform

Usage:  T2AbsoluteToObserver


T2ObserverToAbsolute:  rank-2 tensor transform (inverse)

Usage: T2ObserverToAbsolute


cubiC:  elastic constants of cubic crystal in arbitrary frame

Purpose: suppose the crystal has cubic symmetry with
elastic constants c11,c12,c44 in the current frame,
print out 6x6 elastic constant matrix in any new frame.

Usage: cubiC c11 c12 c44
